# Пошаговая инструкция запуска Telegram бота

## Шаг 1: Проверка зависимостей

Убедитесь, что установлены все зависимости:

```bash
.\venv\Scripts\Activate.ps1
pip install python-telegram-bot
```

## Шаг 2: Проверка токена

Откройте файл `.env` в корне проекта и убедитесь, что там указан токен:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
TELEGRAM_BOT_ENABLED=true
```

**Как получить токен:**
1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 3: Проверка конфигурации

Запустите проверку:

```bash
.\venv\Scripts\Activate.ps1
python check_bot.py
```

Должно вывести:
```
✅ Все проверки пройдены! Бот готов к запуску.
```

## Шаг 4: Запуск бота

### Вариант 1: Простой запуск

```bash
.\venv\Scripts\Activate.ps1
python start_bot.py
```

### Вариант 2: Оригинальный скрипт

```bash
.\venv\Scripts\Activate.ps1
python run_telegram_bot.py
```

## Шаг 5: Проверка работы

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте команду `/start`
4. Бот должен ответить приветственным сообщением

## Возможные проблемы

### Бот не отвечает

1. **Проверьте, что бот запущен:**
   ```powershell
   Get-Process python | Where-Object {$_.Path -like "*venv*"}
   ```

2. **Проверьте логи** - в консоли должны быть сообщения без ошибок

3. **Проверьте токен** - убедитесь, что токен правильный и не содержит лишних пробелов

4. **Проверьте интернет** - бот должен иметь доступ к интернету

### Ошибка "python-telegram-bot не установлен"

```bash
pip install python-telegram-bot
```

### Ошибка "TELEGRAM_BOT_TOKEN не установлен"

Проверьте файл `.env`:
- Токен должен быть на одной строке
- Не должно быть кавычек вокруг токена
- После `=` должен быть токен без пробелов

### Бот запускается, но не отвечает

1. Убедитесь, что вы пишете правильному боту (проверьте имя бота)
2. Попробуйте отправить `/start` еще раз
3. Проверьте, что бот не заблокирован
4. Посмотрите логи в консоли - там могут быть ошибки

## Запуск в фоновом режиме (Windows)

Для запуска в фоне используйте:

```powershell
Start-Process python -ArgumentList "start_bot.py" -WindowStyle Hidden
```

Или через Task Scheduler для автозапуска.

## Остановка бота

Нажмите `Ctrl+C` в консоли, где запущен бот.

Или завершите процесс:

```powershell
Get-Process python | Where-Object {$_.Path -like "*venv*"} | Stop-Process
```
